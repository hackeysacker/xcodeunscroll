# Beta Testing Plan

## Scope
- Authentication (sign in/sign out)
- Challenge completion flow
- Gems/hearts progression
- Notifications behavior
- Offline/online transitions

## Test Matrix
- iPhone 16 / latest iOS
- iPhone 17 Pro / latest iOS
- One iPad model (portrait + landscape)

## Feedback Template
- Device + OS
- Build number
- Repro steps
- Expected vs actual
- Screenshot/video

## Gates
- No blocker crashes
- No data-loss bugs
- Onboarding and challenge loops pass end-to-end
