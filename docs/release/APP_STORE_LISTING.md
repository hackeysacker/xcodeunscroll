# App Store Listing Checklist

## Metadata
- App name: Unscroll
- Subtitle
- Description (short + long)
- Keywords
- Support URL
- Privacy policy URL
- Marketing URL (optional)
- Metadata templates live in `fastlane/metadata/en-US`
- Upload metadata with `fastlane ios metadata`

## Assets
- App icon (1024x1024)
- iPhone screenshots (6.9, 6.7, 6.5, 6.1)
- iPad screenshots
- App preview videos (optional)

## Compliance
- Age rating questionnaire
- Data collection/privacy labels
- Encryption declaration

## Release
- Select phased release or manual release
- Add release notes
- Submit for review
